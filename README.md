VibeCheck API + Button Smash UI
A small full-stack project that combines a basic Express backend with a simple button-based frontend interface.

What This Project Includes
Backend (Node.js + Express)

-A server that returns JSON responses.

-Routes that generate fortunes, jokes, vibe messages, and a running smash counter.

-A secret route that reveals a hidden message when the correct code is provided.

Frontend (HTML + JavaScript)
-Buttons that call the backend routes using fetch().
-A display area that shows the returned JSON.
-A SMASH button linked to the counter endpoint.

How to Use
1. Install dependencies inside the backend folder.
2. Run the backend server.
3. Open the frontend HTML file in your browser.
4. Click the buttons to trigger API calls and view results.
